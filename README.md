# Ecommerce-Customer-Churn-Predictor
An AI-powered E-Commerce analytics dashboard that uses Machine Learning (RFM analysis) to predict customer churn probability and estimate Customer Lifetime Value (CLV).
## 🌍 Live Demo
Click the link below to view the interactive live website:
👉 [Visit Live Website](https://e-commerce-analytics-rxy4.bolt.host)

## ✨ Key Features
* **Predictive Churn Engine:** Uses Machine Learning logic based on RFM (Recency, Frequency, Monetary) analytics to calculate churn risks instantly.
* **Customer Lifetime Value (CLV) Forecasting:** Segments customers into groups (VIP, At Risk, New) for targeted marketing.
* **Interactive Business Dashboard:** A clean, modern SaaS-style UI featuring dynamic sales and customer metrics charts.

## 🛠️ Tech Stack
* **Frontend/UI:** React.js / Tailwind CSS (via Bolt.new)
* **Data & Analytics Logic:** Python ecosystem (Pandas, Scikit-Learn logic)

Built as part of my preparation for a Master's in Data Science in France. It demonstrates the practical application of data structures and machine learning algorithms to solve real-world business challenges.
